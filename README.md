# Grafo 

Um grafo é uma estrutura de dados composta por um conjunto de vértices (nós) e arestas (ligações) que representam relações entre elementos, sendo amplamente utilizado para modelar problemas como redes, caminhos e conexões. Ele pode ser direcionado ou não direcionado, ponderado ou não, dependendo da natureza das relações, e permite a aplicação de algoritmos para busca, caminhos mínimos e análise de conectividade, tornando-se fundamental em áreas como computação, logística e sistemas complexos.
